# 웹 크래프트 워 (Web Craft War) — 게임 기획 초안

> **출처:** viral-game-idea 크론잡 수집 (2026-02-21)

---

## 한 줄 소개

웹 브라우저 기반 실시간 전략 시뮬레이션(RTS) — HTML·CSS·JS 리소스를 채집해 유닛을 생산하고 다른 유저와 전투

---

## 벤치마크

| 항목 | 내용 |
|---|---|
| **참고 게임** | Crazy Web Multiplayer Jam 2025 + 실시간 전략 시뮬레이션 (RTS) |
| **장르** | 브라우저 기반 멀티플레이어 RTS |

---

## 핵심 메카닉

- **리소스 채집:** 웹 페이지를 '해킹'하여 HTML / CSS / JavaScript 리소스 수집
- **유닛 생산:** 채집 리소스로 기본 유닛 생산 및 업그레이드
  - HTML 병사 / CSS 마법사 / JavaScript 해커
- **전투:** 상대 웹 페이지 공격→리소스 탈취, 내 페이지 방어→리소스 보호

---

## 중독성 요소

- 실시간 전황 변화에 따른 전략적 판단 요구
- 다른 유저와 연합 결성 및 대규모 전투

---

## 바이럴 포인트

- 전투 결과·전략 소셜 미디어 공유
- 게임 내 웹 페이지 홍보 및 방문 시스템

---

## MVP 범위

1. 기본 리소스 채집 시스템
2. 3종 기본 유닛 (HTML / CSS / JS)
3. 간단한 전투 시스템
4. 웹 페이지 점령 시스템
5. 랭킹 시스템
6. 채팅 기능
